

## Description

Challenger Coink - App de registro
- Ionic
- Angular

Deberan crear un .env con las siguientes variables



## Installation

```bash
$ npm install -g @ionic/cli
```

## Running the app

```bash
# Local Web
$ npm start

# Local IOS + Generate Build
$ npm run start:ios

# Local Android + Generate Build
$ npm run start:android
```
